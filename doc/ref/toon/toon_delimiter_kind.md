### jsoncons::toon::toon_delimiter_kind

```cpp
#include <jsoncons_ext/toon/toon_options.hpp>

enum class toon_delimiter_kind : char
{
    comma = ',, 
    tab = '\t', 
    pipe = '|'
};
```

Value      |Definition
-----------|-----------
comma      | Comma delimiter (default)
tab        | Tab delimiter
pipe       | Pipe delimiter
